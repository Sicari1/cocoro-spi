# Cocoro SPI

SPI preparation web app for Japanese job seekers.

## Stack

- Frontend: HTML, CSS, vanilla JavaScript
- Backend: Python `http.server` + SQLite
- Port: `8080`

## Run

```bash
bash /home/seongjin/Cocoro/serve.sh
```

Open:

```text
http://localhost:8080
```

## Features

- Practice tab with explanations and answer feedback
- Searchable question database
- Mock exam mode with timer and review
- Score tracking backed by SQLite
- Cinnamoroll-inspired themed UI
